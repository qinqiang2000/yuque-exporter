# 飞书知识库同步使用指南

## 前置条件

### 1. 构建项目

```bash
npm run build
```

### 2. 准备飞书应用凭证

需要一个飞书自建应用的 App ID 和 App Secret。获取方式：
1. 打开[飞书开放平台](https://open.feishu.cn)
2. 创建或选择已有自建应用
3. 在「凭证与基础信息」页面获取 App ID 和 App Secret
4. 确保应用已开通「wiki 知识库」相关权限

---

## 基本用法

```bash
node dist/bin/cli.js feishu <node_token> -o <输出目录> \
  --feishu-app-id <App ID> \
  --feishu-app-secret <App Secret>
```

或通过环境变量传入凭证：

```bash
FEISHU_APP_ID=cli_xxx FEISHU_APP_SECRET=yyy \
  node dist/bin/cli.js feishu <node_token> -o ./data/feishu
```

### 获取 node_token

飞书 wiki 页面 URL 的最后一段即为 node_token：

```
https://xxx.feishu.cn/wiki/W3IzwdlAWisu3fkJ2V7c3HVGnhd
                                ^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                这就是 node_token
```

也可以直接传入完整 URL，程序会自动提取 token。

---

## 下载文件附件

飞书的文件附件（.xls、.pdf 等）不支持应用 token 下载，需要额外提供浏览器登录 Cookie。

### 获取 Cookie

1. 浏览器打开飞书并登录
2. 打开开发者工具（F12）→ Network 标签
3. 刷新页面，点击任意请求
4. 在请求头里找到 `Cookie` 字段，复制完整内容

### 传入 Cookie

```bash
FEISHU_APP_ID=cli_xxx FEISHU_APP_SECRET=yyy \
  FEISHU_COOKIE='your_cookie_here' \
  node dist/bin/cli.js feishu <node_token> -o ./data/feishu
```

或通过 flag：

```bash
node dist/bin/cli.js feishu <node_token> -o ./data/feishu \
  --feishu-app-id cli_xxx \
  --feishu-app-secret yyy \
  --feishu-cookie 'your_cookie_here'
```

---

## 增量同步

重复运行同一命令即可增量同步，只处理自上次同步后有变更的文档，未变更的文档会跳过。

---

## 同步多个知识空间

依次运行多条命令，输出到同一目录：

```bash
node dist/bin/cli.js feishu <token1> -o ./data/feishu
node dist/bin/cli.js feishu <token2> -o ./data/feishu
node dist/bin/cli.js feishu <token3> -o ./data/feishu
```

---

## 清空重建

加 `--clean` 会在同步前清空输出目录：

```bash
node dist/bin/cli.js feishu <node_token> -o ./data/feishu --clean
```

---

## 只爬取 / 只构建

```bash
# 只爬取（拉取 API 数据缓存到 .meta/，不生成 Markdown）
node dist/bin/cli.js feishu crawl <node_token> -o ./data/feishu

# 只构建（从已有缓存生成 Markdown，不调用 API）
node dist/bin/cli.js feishu build <node_token> -o ./data/feishu
```

---

## 完整选项

| 选项 | 环境变量 | 说明 |
|------|----------|------|
| `--feishu-app-id` | `FEISHU_APP_ID` | 飞书应用 App ID |
| `--feishu-app-secret` | `FEISHU_APP_SECRET` | 飞书应用 App Secret |
| `--feishu-cookie` | `FEISHU_COOKIE` | 浏览器 Cookie，用于下载文件附件 |
| `-o, --output` | — | 输出目录，默认 `./storage` |
| `--clean` | — | 同步前清空输出目录 |
| `--timeout` | — | 请求超时时间（毫秒），默认 `60000` |
| `-c, --concurrency` | — | 并发请求数，默认 `10` |
| `--max-retries` | — | 失败重试次数，默认 `3` |
| `--retry-delay` | — | 重试间隔（毫秒），默认 `1000` |
